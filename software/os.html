mizOS

License: GNU GPL v.3

Maintainer: Mizosu
Source: github.com/Mizosu97/mizOS

mizOS is an Arch Linux-based operating system written in a combination of Lua and Bash. It aims to provide a simple, easy-to-use system while still maintaining a fast and lightweight user experience.
