Contributing to the Duat

Fully free, transluscent software



How do I contrubute software?

Contributing software to the Duat is relatively easy. You are not limited to your own software, you can make a request to list a piece of software you do not own/did not make.



Software Requirements

The software must comply with the following:

- Software must have clear, well-written docunentation.*

- Software's main source code must be hosted on a git website (Github, Gitlab, Codeberg, etc.)

- Software must be void of any monetization methods, with the specific exception of donations, advertisements, and sponsorships.

- The software must be licensed under any of the following licenses:

> GNU GPL 3.0

> GNU LGPL 3.0

> MIT License

> 3-Clause BSD License

> 2-Clause BSD License

> Apache License 2.0

> WTFPL

> DBAD License


Once you have verified that the software meets these requirements, you can proceed to the section below.



Making the request

Please contact the site owner with the following information:

- Name of the software.

- A short (up to 25 words) description of the software.

- A link to the software's source code. (git repository)

- A personal identification link of the software's main developer. (Github/Gitlab/Codeberg/etc profile will suffice.

- What license the software uses.


Contact information is listed on the right-hand side of this website. You will be notified via your chosen contact information of your contribution's status.
